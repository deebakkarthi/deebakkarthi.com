| Type                     | Naming Convention                  |
| ------------------------ | ---------------------------------- |
| Package and Module Names | lowercase                          |
| Class Names              | PascalCase                         |
| Type Variables           | PascalCase                         |
| Exceptions               | PascalCase(Exceptions are a class) |
| Functions                | snake_case                         |
| Constants                | UPPERCASE                          |
